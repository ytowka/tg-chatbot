# Чат бот для телеграма

## Функциональные требования
- Чтобы чат бот заработал, он должен быть участником группы в телеграм. 
- Его задача общаться с другими участниками и отвечать на их вопросы. Бот должен откликаться на сообщения в которых есть его упоминание (фраза для отклика настраиваемая), или если отвечают на его сообщение  
- Среди возможностей должна быть возможность выдавать ответы на основе всей истории чата. 
- По запросу делать краткую сводку предыдущего дня

## Работа с контекстом
- Контекст накапливается со всеми сообщениями с начала первого взаимодействия с ботом
- Если по прошествии N часов не было взаимодействий с ботом, контекст очищается
- Должен быть файлик с памятью бота, в которой хранится компактная информация о чате и участниках, накапливаемая в ходе диалогов. Файл обновляется перед очищением контекста 
- Должна быть команда для очистки контекста принудительно

## Защита от опасных действий
- Бот не может удалять сообщения, удалять участников и другие подобные действия
